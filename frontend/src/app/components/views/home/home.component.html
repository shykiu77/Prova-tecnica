Bem Vindo!

Este é um sistema crud desenvolvido por Carlos Gabriel Santos de Melo para a a prova técnica da ma9.

Funcionalidades do sistema:
  • Cadastro de clientes ao servidor através do botão Adicionar cliente na aba clientes
  • Edição dos dados dos clientes
  • Remoção de clientes, acompanhado de um pop-up de confirmação
  • Opção para adiconar clientes a uma lista de favoritos, sendo possível também remove-los através de um click
  • Destaque de qual funcionalidade esta sendo usada atualmente através de uma mudança na cor dos ícones
